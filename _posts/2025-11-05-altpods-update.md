AltPods v1.54.0 were released to Maven central. New scan performed to sync with recent releases and available for testing as v1.55.0-SNAPSHOT.


### Updated pods (ones with API changed)
- [AppLovinSDK](https://github.com/dkimitsa/robovm-robopods/tree/dev/v1.55.0/applovinsdk/)  `v13.4.0` -> `v13.5.0`
- [CleverAds](https://github.com/dkimitsa/robovm-robopods/tree/dev/v1.55.0/cleverads/)  `v4.3.0` -> `v4.4.2`
- [OneSignal](https://github.com/dkimitsa/robovm-robopods/tree/dev/v1.45.0/onesignal/)  `v5.2.14` -> `v5.2.15`
- [Singular](https://github.com/dkimitsa/robovm-robopods/tree/dev/v1.45.0/singular/)   `v12.8.1` -> `v12.9.0`

* These pods were pushed to `https://central.sonatype.com/repository/maven-snapshots/` maven repo under `1.55.0-SNAPSHOTS` version.
[Source code @github](https://github.com/dkimitsa/robovm-robopods/tree/dev/v1.54.0)

Updates are not fully tested, please [open issue](https://github.com/dkimitsa/robovm-robopods/issues/new) if bug found.
